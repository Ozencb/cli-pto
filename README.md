# cli-pto

![](Gif.gif)

## Description
**cli-pto** is a CLI text editor with encryption

## Installation
Run `python setup.py install` to install cli-pto

## Usage
Simply run `cli-pto` to run. You can provide a file name as an argument to skip file dialog
